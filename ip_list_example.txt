# 3proxy IP Liste Örneği
# Bu dosya proxy oluşturma için gereken IP adreslerini içerir
# Her satırda bir IP adresi olmalıdır
# IP'leri subnet ile birlikte girebilirsiniz (önerilen)
# Boş satırlar ve # ile başlayan satırlar yok sayılır

# Örnek IPv4 adresleri subnet ile (Kendi IP'lerinizle değiştirin)

# Subnet olmadan da girebilirsiniz (otomatik tespit edilir)

10.0.0.50
172.16.1.10

# NOTLAR:
# - Subnet ile girilen IP'ler daha güvenilir
# - /23 subnet = 512 IP (örn: 45.90.98.0-45.90.99.255)
# - /24 subnet = 256 IP (örn: 176.100.39.1-176.100.39.254) 
# - /22 subnet = 1024 IP (örn: 185.14.92.0-185.14.95.255)
# - Her proxy için bir IPv4 adresi gereklidir
# - IP adresleri sunucuda mevcut olmalıdır
# - Maksimum mod için her IP'den 3 proxy oluşturulur
# - Bu örnekte 10 IP = maksimum 30 proxy oluşturulabilir
